# CSS基础

## 什么是CSS？

> CSS（Cascading Style Sheets，层叠样式表）是用于设计网页样式的语言。它就像是我们给网页"化妆"的工具，可以控制网页的颜色、布局、字体等外观特征。CSS让我们的网页从简单的文字变成了精美的艺术品。

### 1. 引入CSS的方式

在HTML中，我们有三种方式可以引入CSS样式：

1. 内联样式：直接在HTML元素中使用style属性
   ```html
   <p style="color: red; font-size: 16px;">这是红色文字</p>
   ```

2. 内部样式表：在`<head>`标签中使用`<style>`标签
   ```html
   <head>
     <style>
       p { color: red; font-size: 16px; }
     </style>
   </head>
   ```

3. 外部样式表：使用`<link>`标签引入外部CSS文件
   ```html
   <head>
     <link rel="stylesheet" href="styles.css">
   </head>
   ```

### 4. CSS选择器

CSS选择器就像是我们的"定位器"，帮助我们找到想要美化的HTML元素。就像在人群中找人一样，我们需要知道这个人的特征（ID、类名、标签名等）。

让我们通过一个具体的例子来理解选择器：

```html
<div id="container">
  <div class="header">
    <h1 class="title">标题</h1>
    <p class="subtitle">副标题</p>
    </div>
  <div class="content">
    <p>第一段文本</p>
    <p class="highlight">高亮文本</p>
    <p>第三段文本</p>
    </div>
  <div class="footer">
    <span>页脚信息</span>
  </div>
</div>
```

这个HTML结构可以用树状图表示：

```
div#container
├── div.header
│   ├── h1.title（标题）
│   └── p.subtitle（副标题）
├── div.content
│   ├── p（第一段文本）
│   ├── p.highlight（高亮文本）
│   └── p（第三段文本）
└── div.footer
    └── span（页脚信息）
```

现在让我们看看如何使用不同的选择器来选择这些元素：

```css
/* ID选择器：选择具有特定ID的元素 */
#container {
  width: 800px;
  margin: 0 auto;
}

/* 类选择器：选择具有特定类名的元素 */
.header {
  background-color: #f0f0f0;
  padding: 20px;
}

/* 元素选择器：选择特定类型的元素 */
p {
  line-height: 1.6;
  margin-bottom: 10px;
}

/* 后代选择器：选择div内的所有p元素 */
.content p {
  color: #333;
}

/* 子元素选择器：选择.header的直接子元素 */
.header > h1 {
  color: #2c3e50;
}

/* 相邻兄弟选择器：选择紧跟在.highlight后面的p元素 */
.highlight + p {
  font-style: italic;
}

/* 通用兄弟选择器：选择.highlight后面的所有p元素 */
.highlight ~ p {
  color: #666;
}
```

选择器类型说明：
- ID选择器（#id）：选择具有特定ID的元素，如 `#container`
- 类选择器（.class）：选择具有特定类名的元素，如 `.header`
- 元素选择器（element）：选择特定类型的元素，如 `p`
  - 通配符选择器（*）：选择所有元素
  - 后代选择器（div span）：选择div内的所有span元素
  - 子元素选择器（div > span）：选择div的直接子元素span
  - 相邻兄弟选择器（div + span）：选择紧跟在div后面的span元素
  - 通用兄弟选择器（div ~ span）：选择div后面的所有span元素

### 5. 伪类选择器

伪类选择器用于定义元素的特殊状态。它们就像是元素的"情绪"，可以让我们根据元素的状态来应用不同的样式。

<Canvas of={CSSStories.PseudoClasses} />

#### 常用的伪类选择器：

1. 用户交互伪类
   - `:hover` - 当鼠标悬停在元素上时
   - `:active` - 当元素被激活（如点击）时
   - `:focus` - 当元素获得焦点时（如表单输入框）

2. 结构伪类
   - `:first-child` - 选择父元素的第一个子元素
   - `:last-child` - 选择父元素的最后一个子元素
   - `:nth-child(n)` - 选择父元素的第n个子元素
   - `:nth-child(odd)` - 选择奇数位置的子元素
   - `:nth-child(even)` - 选择偶数位置的子元素

#### 伪类选择器的使用示例：

```css
/* 悬停效果 */
button:hover {
  background-color: #4a90e2;
  color: white;
}

/* 点击效果 */
button:active {
  background-color: #357abd;
  transform: scale(0.95);
}

/* 焦点效果 */
input:focus {
  outline: none;
  border-color: #4a90e2;
  box-shadow: 0 0 5px rgba(74, 144, 226, 0.3);
}

/* 第一个子元素 */
p:first-child {
  color: #4a90e2;
  font-weight: bold;
}

/* 最后一个子元素 */
p:last-child {
  color: #e74c3c;
  font-weight: bold;
}

/* 奇数位置的子元素 */
p:nth-child(odd) {
  background-color: #f8f9fa;
}

/* 偶数位置的子元素 */
p:nth-child(even) {
  background-color: #f0f0f0;
}
```

#### 伪类选择器的使用技巧：

1. 组合使用
   - 可以将伪类选择器与其他选择器组合使用
   - 例如：`.button:hover`、`input:focus`

2. 链式使用
   - 可以连续使用多个伪类
   - 例如：`button:hover:active`

3. 响应式设计
   - 伪类选择器常用于创建交互效果
   - 可以结合`transition`属性创建平滑的过渡效果

4. 表单样式
   - 使用`:focus`伪类美化表单元素的焦点状态
   - 使用`:checked`伪类自定义复选框和单选框的样式

5. 列表样式
   - 使用`:first-child`和`:last-child`为列表的首尾项添加特殊样式
   - 使用`:nth-child`创建斑马纹效果

### 8. 布局

CSS提供了强大的布局系统，其中Flexbox和Grid是最现代和灵活的布局方式。让我们来详细了解这两种布局方式：

#### Flexbox布局

Flexbox（弹性盒子）是一种一维布局模型，主要用于在单个轴（行或列）上排列元素。它特别适合创建响应式的导航栏、卡片布局等。

主要属性：
- `display: flex` - 启用flex布局
- `flex-direction` - 设置主轴方向（row/column）
- `justify-content` - 主轴对齐方式
- `align-items` - 交叉轴对齐方式
- `flex-wrap` - 是否换行
- `gap` - 元素间距

<Controls of={CSSStories.Layout} />

#### Grid布局

Grid（网格）是一种二维布局系统，可以同时处理行和列。它非常适合创建复杂的网页布局，如仪表板、图片画廊等。

主要属性：
- `display: grid` - 启用grid布局
- `grid-template-columns` - 定义列宽
- `grid-template-rows` - 定义行高
- `gap` - 网格间距
- `grid-area` - 定义网格区域
- `grid-column/row` - 控制元素在网格中的位置

<Controls of={CSSStories.Grid} />

布局技巧：
- Flexbox适合一维布局，如导航栏、列表等
- Grid适合二维布局，如页面整体结构、卡片网格等
- 两种布局可以结合使用，例如在Grid的单元格内使用Flexbox
- 使用gap属性可以轻松控制元素间距
- 响应式设计时，可以结合媒体查询调整布局
