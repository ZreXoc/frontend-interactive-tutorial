# 待办事项应用示例

这个示例展示了一个现代化的待办事项（Todo List）应用界面。它不仅展示了CSS布局技巧，还为后续添加JavaScript交互功能做好了准备。

## 界面结构

这个待办事项应用包含以下主要部分：

1. 头部区域
   - 标题
   - 任务统计（待完成/已完成）

2. 输入区域
   - 文本输入框
   - 添加按钮

3. 过滤区域
   - 全部/待完成/已完成 过滤按钮

4. 任务列表
   - 任务项（包含复选框、文本、编辑和删除按钮）
   - 已完成任务的样式变化

## Flexbox布局的应用

在这个示例中，我们大量使用了Flexbox布局来实现各种布局需求：

### 1. 整体布局
```css
.todo-app {
  max-width: 600px;
  margin: 2rem auto;
}
```
- 使用`max-width`限制最大宽度
- 使用`margin: auto`实现水平居中

### 2. 头部区域
```css
.todo-header {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
```
- 使用`justify-content: space-between`实现标题和统计的两端对齐
- 使用`align-items: center`实现垂直居中对齐

### 3. 输入区域
```css
.todo-input {
  display: flex;
  gap: 1rem;
}
```
- 使用`flex`布局实现输入框和按钮的排列
- 使用`gap`属性设置间距

### 4. 任务列表
```css
.todo-list {
  display: flex;
  flex-direction: column;
  gap: 1rem;
}
```
- 使用`flex-direction: column`创建垂直列表
- 使用`gap`属性设置任务项之间的间距

## 交互效果

这个示例包含了丰富的交互效果：

1. 输入框焦点效果
```css
.todo-input input:focus {
  outline: none;
  border-color: #3498db;
}
```

2. 按钮悬停效果
```css
.add-btn:hover {
  background-color: #2980b9;
}
```

3. 任务项悬停效果
```css
.todo-item:hover {
  transform: translateX(5px);
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
}
```

4. 操作按钮的显示/隐藏
```css
.todo-actions {
  opacity: 0;
  transition: opacity 0.3s ease;
}
.todo-item:hover .todo-actions {
  opacity: 1;
}
```
